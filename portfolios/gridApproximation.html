Grid-Based Approximation

Last worked on in November 2021

This project was created as algorithmic practice. The problem statement I set out to solve was as follows:

Given a set of N points (xi, yi) with integer values for xi between 1 and M and real values for yi, find a partitioning of the interval [1, M] into contiguous intervals such that the error of approximating points in each interval element (induced by the partitioning on X axis) by the average value of y in the interval element is minimized. Include a penalty factor proportional to the total number of intervals the solution has.

I solved this problem with two methods: a tabulation approach and a memoization approach. For both methods, dyanmic programming was used. The first thing done was to sort the points with a Merge Sort, and then the error values of all points was precomputed such that the error of any interval in the set can be looked up by the algorithm later at constant time.

The tabulation solution used a bottom-up approach: for each point added to the considered set, find the optimal "last segment" based on error costs and the penalty of adding another interval. This works since the points have been sorted, so the latest considered point is the most rightward point in the considered set. If we know the optimal "last segment" for this point begins at Xj, we must only add this optimal segment to the optimal segment ending at Xj, and so on.

The memoization solution used a top-down approach: we still want to find the optimal "last segment" based on the considered set, but using recursion and a map of numbers of points to their optimal values, computations are only made when they are both necessary and previously uncompleted.
